Usage:

1. Before using you need to decompress the ZIP format file to get a folder named "Software_group22". (The program can be activated by cmd windows)

2. Open the folder named “java”. Then input "cmd" in the address bar of the folder and then press "enter" button to enter the command lines.

3. Compile all java file. Tap in "javac *.java" and then press "enter" button in the cmd window.

4. Running:
 Tap in "java Main" to run the program and then the user interface will show up.

5. Continue running: Open a new cmd window(input "cmd" in the address bar of the folder and then press "enter" button). Then tap in "java Clock" and then press "enter" button in the cmd window to run the clock program(you do not need to operate in this cmd window). The report will be automatically sent to the administrator at 24:00 if the Clock is continue running at that time.

6. Now we finish installing, compiling and running the program.


7. Now you can operate in the user interface.
